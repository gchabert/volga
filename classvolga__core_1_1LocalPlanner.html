Detailed Description

Local planner.

This class maximizes the displacement of a robot in a fixed direction (in either joint or Cartesian space) such that no collision occurs.

The robot and obstacle geometries are defined by triangle meshes.

Build the local planner.

The local planner will find the maximal displacement of the mesh (in argument) without collision along a given curve in SO(3).

Computes the maximal t in [0,1] such that no collision occurs when the mesh pose is x(t) interpolating x0 and x1.

Parameters
link- object (collision) geometry
obstacle- obstacle geometry (single ball so far)
x0- initial pose of the object
x1- final pose of the object

Build the local planner.

The local planner will find the maximal movement of the robot such that the link (given in argument) is collision-free with the obstacle (given in argument).

Computes the maximal t in [0,1] such that no collision occurs when the robot configuration is q(t).

Parameters
robot- the robot
link- link number
space- JOINT or CARTESIAN, see SpaceType.
obstacle- obstacle geometry (single ball so far)
q0- initial configuration of the robot
q1- final configuration of the robot

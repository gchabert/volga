Detailed Description

Tree-structured mesh.

The mesh is defined by a classical set of triangle facets and a tree structure of balls representing different levels of approximation of the geometry. This allows fast collision computation.

Build the mesh from a list of vertices/triangles.

Parameters
vertex_count- The number of available vertices
vertices- The position for each vertex. Vertex k has values at index (3k, 3k+1, 3k+2) = (x,y,z)
triangle_count- The number of triangles formed with the vertices
triangles- The vertex indices for each triangle. Triangle k has vertices at index (3k, 3k+1, 3k+2) = (v1, v2, v3)
meters- True if the unit is meter (false=millimeter)
eps_meters- the precision in meters the mesh must be refined with (the precision is in meters even if meters=false).
log- Verbose if true

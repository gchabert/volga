Detailed Description

Interpolating function for a frame in SO(3).

Useful for local planning in Cartesian space.

This class also contains the Ibex system (constraints) representing the collision of two balls, one in the interpolated frame and the other one fixed.

The class uses both spherical and linear interpolation for the rotation, depending on initial/final frames.

The class resorts to mutable constants (references) so that they can be dynamically updated. This way, a single instance of the function/system is created (=> static).

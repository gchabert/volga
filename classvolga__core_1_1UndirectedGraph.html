Detailed Description

template<typename VertexData>
class volga_core::UndirectedGraph< VertexData >

Undirected graph.

This is a simple home-made implementation of undirected graph for faster facet merging algorithm (that is, not in O(n^2)). Existing open-source libraries (like the Boost Graph Library) does not provide such functionality apparently because there is no generic behavior to handle the "corner cases"
